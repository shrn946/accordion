# Flex Accordion for Elementor

Flex Accordion is a powerful and lightweight WordPress plugin that adds a highly customizable, menu-powered accordion widget to Elementor. It allows you to transform any WordPress navigation menu into a sleek, vertical accordion with modern UI features like interactive accent bars and preloaded color patterns.

![Flex Accordion Preview](screen.jpeg)

## Features

- **Menu-Powered:** Dynamically pull items from any WordPress navigation menu.
- **Preloaded Styles:** Choose from 6 professionally designed color patterns:
  - Clean Light
  - Royal Blue
  - Sunset Glow
  - Forest Green
  - Midnight Purple
  - Minimalist Grey
- **Full Customization:** Manually override colors, typography, spacing, and icons through Elementor's style tab.
- **Interactive Design:** Smooth height transitions and animated accent bars for a premium feel.
- **Responsive:** Fully mobile-friendly and lightweight.
- **Developer Friendly:** Clean code structure using namespaces and Elementor best practices.

## Installation

1. Download the plugin folder as a ZIP file.
2. In your WordPress dashboard, go to **Plugins > Add New**.
3. Click **Upload Plugin** and select the ZIP file.
4. Click **Install Now** and then **Activate**.

## Usage

1. Open any page or post with the Elementor Editor.
2. Search for the **Flex Accordion** widget in the sidebar.
3. Drag and drop it into your layout.
4. Under the **Content** tab, select the WordPress menu you want to display.
5. Go to the **Style** tab to choose a **Preloaded Style** or customize every detail manually.

## Requirements

- WordPress 6.0 or higher
- PHP 7.4 or higher
- Elementor (Free or Pro)
